# Project Template

## Requirements

- Python 8+
- SQLite, MySQL or PostgreSQL

## Installation

### Set up virtual environment

```shell
python -m venv venv
source venv/bin/activate
```

### Install dependencies

```shell
pip install -r requirements/dev.txt
```

### Install `pre-commit` hooks

- Install `pre-commit` hooks:

  ```shell
  pre-commit install
  ```

## Running

```shell
python manage.py
```
